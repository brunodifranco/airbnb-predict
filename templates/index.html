Airbnb Destinations Predictions

This is part of the Airbnb Classification Project.

What does it do?

  • Loads Airbnb sessions and users data from a PostgreSQL database
  • Transforms the data and creates predictions by using a Machine Learning model
    • Obs: The model predicts the five most likely countries for an Airbnb user to book their next destination.
  • Loads back the new data (with predictions) into the PostgreSQL database, so it can be recovered by the Streamlit app.
    • Feel free to access the Streamlit app here.

Why is it useful?

  • If new raw data gets added in the Database it's quite easy to get new predictions.

How to use it?

  • Click on the 'Predict' button below, then after a couple of minutes a message should appear on the screen, showing the operation was completed.
